# 🍓 Recipe Studio

A beautiful, responsive recipe management web application built using **HTML5, CSS3, and Vanilla JavaScript**.

Recipe Studio helps users explore recipes, save favorites, plan weekly meals, generate shopping lists, and manage cooking time with a clean pastel-inspired interface.

---

## ✨ Features

### 🔍 Recipe Explorer
- Search recipes instantly
- Filter recipes by category:
  - All
  - Breakfast
  - Lunch
  - Dinner
  - Dessert
- Beautiful recipe cards with smooth animations

---

### 📖 Recipe Details
- Dynamic recipe modal
- View:
  - Ingredients
  - Cooking steps
  - Cooking duration
- Minimal cooking timer included

---

### ❤️ Favorites System
- Add/remove favorite recipes
- Favorites stored permanently using:
- localStorage



---

### 📅 Weekly Meal Planner

Plan your meals for:

- Monday
- Tuesday
- Wednesday
- Thursday
- Friday
- Saturday
- Sunday


Features:
- Assign recipes to days
- Automatically save meal plans
- Persistent user data
